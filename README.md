# 📊 HR Salary Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive HR Salary Analysis Dashboard built entirely in **Microsoft Excel**. It provides valuable insights into employee salary data through dynamic visualizations, Pivot Tables, Charts, Slicers, and KPIs, enabling users to analyze compensation trends efficiently.

---

## 🎯 Objectives

- Analyze employee salary distribution.
- Compare salaries across different locations.
- Monitor key HR metrics.
- Provide an interactive dashboard for quick decision-making.
- Transform raw HR data into meaningful business insights.

---

## 📈 Dashboard Features

- 📌 KPI Cards
  - Total Salary
  - Average Salary
  - Employee Count
  - Other key HR metrics

- 📊 Interactive Charts
  - Salary by Location
  - Salary Distribution
  - Department Analysis
  - Job Level Analysis
  - Additional HR visualizations

- 🎛️ Interactive Filters
  - Slicers

- 📍 Dynamic Dashboard
  - All charts update automatically based on selected filters.
